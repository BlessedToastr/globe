Globe is a very simple port scanner built in GO and faster than nmap

## Quick start
Download the release that corresponds to your system
To Run:
### Linux
```
globe -i [ip]
```

### Windows
```
globe.exe -i [ip]
```

To run in all port mode:
```
globe -i [ip] -p all
```
